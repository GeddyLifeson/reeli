# 🎬 Reeli

**Beli, but for movies.** Rank everything you've ever seen in head-to-head matchups — no star ratings, every score is earned.

## ▶️ [Open the app → geddylifeson.github.io/reeli](https://geddylifeson.github.io/reeli/)

*(You're looking at the source code. The link above is the actual app.)*

- ⚔️ Head-to-head ranking: "Which did you like more?" builds your 0–10 scored list
- 🌍 Live search across a worldwide movie catalog, with real posters
- 🔖 Watchlist, activity feed, and Reelmates (taste-match with friends)
- 🌗 Light/dark themes, keyboard shortcuts, works on phone and desktop
- 🔒 Your data stays in your browser — nothing is sent anywhere

Built with plain HTML/CSS/JS in a single file. Movie metadata and posters from the free [Cinemeta](https://v3-cinemeta.strem.io) catalog.
